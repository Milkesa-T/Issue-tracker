A full-stack **Issue Tracking Web Application** built with **React + TypeScript** on the frontend and **Node.js + Express + MySQL** on the backend.  
The system allows users to create, view, and manage issues with priority and status tracking in a clean, modern UI.

---

##  Features

- Create new issues with **title, description, priority, and status**
- View all issues in a clean **card-based UI**
- Update issue status (Open / In Progress / Closed)
- Persistent data storage using **MySQL**
- RESTful API architecture
- Responsive and portfolio-ready UI
